# FlowAI-Toolkit

## Dependencies

### This project requires the following Python packages:

OpenCV (opencv-python)
TensorFlow (tensorflow)
Keras (keras)
NumPy (numpy)
Matplotlib (matplotlib)
SciPy (scipy)
Scikit-learn (scikit-learn)
Scikit-image (scikit-image)
PySide6 (PySide6)

### Instalation:

pip install opencv-python tensorflow keras numpy matplotlib scipy scikit-learn scikit-image PySide6
